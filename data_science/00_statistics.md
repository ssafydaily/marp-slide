# 기초 통계량

- 데이터를 표현하는 변수들은 무수히 많은 다른 값들을 가진다.
- 그렇다면, 데이터를 처음에 살펴볼 때 어떻게 접근하면 좋은가?

> 데이터 분석의 첫 단계는 데이터의 크기, 분포, 중심, 퍼짐을 파악하기
> 대표값(위치)과 변동성(퍼짐)을 요약하는 통계량 활용
  - 각 변수를 대표하는 값을 구해서 살펴보기
  - 데이터가 얼마나 퍼져 있는지를 확인 

---

# 위치

- 각 변수를 대표하는 값으로 데이터 살펴보기

- 위치를 추정하는 다양한 방법

  - 평균(mean) - 전체적인 중심을 나타내지만 이상치에 민감
  - 절사 평균(trimmed mean) - 상/하위 극단값 일부를 제거해서 이상치 영향을 줄인다.
  - 가중 평균(weighted mean) - 각 값의 중요도(가중치)를 반영해 계산. 값의 영향력이 동일하지 않을 때 사용
  - 중간값(median) - 정렬된 데이터의 중앙 위치 값으로 왜곡된 분포에서도 안정적으로 중심을 나타냄
  - 가중 중간값(weighted median) - 가중치 고려해서 중앙 위치 결정, 표본 비중이 다른 조사에서 활용
  - 백분위수(percentile) - 데이터를 100등분 했을 때 특정 위치의 값으로 분포의 형태와 위치를 파악

--- 

# 변이

- 각 변수를 대표하는 값을 구하여 데이터를 살펴보기 
- 데이터가 얼마나 퍼져 있는지를 확인

- 변이를 추정하는 방법

  - 분산(variance) - 평균으로 부터 얼마나 퍼져 있는지
  - 표준편차(standard deviation) - 데이터의 편차(각 값에서 평균을 뺀 값)의 기준이 되는 값
  - 사분할 볌위(IQR) - 25번째와 75번째 백분위수의 차이를 보는 것

# 기초 통계량 활용

- 이상치 파악
  - 이상치는 평균/분산을 크게 왜곡하므로 확인

- 비교와 예측 
  - 두 개 이상의 그룹을 비교할 때 **어떤 차이** 가 있는지 시각적으로 확인
  - 평균/중앙갑/표준편차를 이용해 집단 특성 파악

- 적절한 분석 방법 선택
  - 데이터의 분포 형태에 따라 **평균, 중앙값 등을 선택** 하여 분석 가능
  - 대칭형 분포라면 평균을 사용, 왜곡된 분포라면 중앙값이 더 적절
