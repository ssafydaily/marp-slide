# 변수

#### 변수 생성

- 변수에 값을 할당하면 변수가 생성 => 메모리 공간 생성
```python

pi = 3.14
# ...
name = '고길동'  # 변수의 값을 변경
```

# 표현식
- 컴퓨터에서 계산작업을 수행하는 것을 평가(evaluation)한다고 한다.
- **표현식** 은 계산을 통해서 하나의 값으로 추출
- 보통 연산자와 피연산자(값)들의 조합으로 표현
- 함수 호출도 표현식이다.

---
# 할당문

- `변수 = 표현식` 의 형태
- `=` 의 오른쪽에 작성된 표현식이 평가된 최종 값이 변수에 할당

```python
val = 10    # 초기값 할당

val = 20    # 재할당

val = val - 5  # =(할당연산자)
```

---
# 표혆식의 예

```python
# 산술식
(money * rate) + 100

# 비교식 => True/False를 나타는 값

left_value > right_value

# 논리식 => True/False

exp1 and exp2

# 함수 호출 > 함수 실행 후 반환값

calc_score()

```
