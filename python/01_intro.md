# 파이썬 자료형
## 기본형 (Scalar Types)
- 가장 기본적인 재료 이것은 더 이상 쪼갤 수 없는 단일 값
- 요리로 치면 설탕, 소금 같은 기본 재료들이죠

>
> - int (정수): 소수점이 없는 숫자 ($1, 0, -5$)
> - float (실수): 소수점이 있는 숫자 ($3.14, -0.5$)
> - str (문자열): 글자들의 집합. 따옴표(', ")로 표기 
> - bool (불리언): 참(True) 아니면 거짓(False)만 나타내는 논리값

----------------------------------
# 파이썬 자료형

### 2. 컨테이너형 (Container Types): "여러 개를 담는 바구니"

1. **리스트 (List) & 튜플 (Tuple)**
    - **리스트 `[]`:** 순서가 있고, 내용을 마음대로 넣었다 뺐다 할 수 있는 **'다용도 수납함'**
    - **튜플 `()`:** 순서는 있지만, 한 번 넣으면 절대 바꿀 수 없는 **'박제된 전시함'**

2. 딕셔너리 (Dictionary)
    - **딕셔너리 `{}`:** '단어:뜻'처럼 **'키(Key):값(Value)'**의 쌍으로 저장하는 **'이름표 붙은 서랍장'**

3. 집합 (Set)
    - **집합 `{}`:** 중복을 허용하지 않고 순서도 없는 **'주머니'**
----------------------------------

# sequence & non-sequence
- 데이터를 관리할 때 **순서(Order)**가 있느냐 없느냐가 매우 중요한 기준

#### 1. 시퀀스 (Sequence) 자료형
- 데이터의 **위치(Index)**가 중요
- **예시:** "문자열은 '문자'들이 줄을 서 있는 시퀀스다!"

#### 2. 비시퀀스 (Non-Sequence) 자료형
- 데이터의 순서보다는 **"특정 데이터가 안에 있는지"** 혹은 **"이 이름표에 해당하는 값이 무엇인지"**가 중요
- 데이터의 존재 여부를 확인하는 속도가 시퀀스보다 압도적으로 빠름
- **예시:** "사전(Dictionary)에서 단어를 찾을 때 색인을 보고 바로 찾는 것과 같다!"

----------------------------------
> **참고 (Professor's Note):** > 파이썬 3.7 버전부터 `dict`는 입력 순서를 내부적으로 기억하기 시작했지만, 여전히 위치 번호(Index)로 접근할 수 없기 때문에 개념적으로는 비시퀀스(매핑형)로 분류

----------------------------------

| 구분 | 시퀀스 (Sequence) | 비시퀀스 (Non-Sequence) |
| :--- | :--- | :--- |
| **해당 자료형** | `list`, `tuple`, `str`, `range` | `dict`, `set` |
| **핵심 개념** | **순서(Order)** 기반 | **해시(Hash)** / **관계** 기반 |
| **접근 방식** | **인덱싱** (`a[0]`) | **키(Key)** 접근 (`a['key']`) 또는 불가능 |
| **범위 추출** | **슬라이싱** (`a[1:3]`) 가능 | **슬라이싱 불가능** |
| **중복 데이터** | 허용 (동일 값 중복 저장 가능) | `set`은 불가, `dict`는 Key 중복 불가 |
| **데이터 순서** | 입력한 순서가 유지됨 | 순서가 유지되지 않음 (Python 3.7+ dict 제외*) |
| **현실 비유** | 출석부, 기차 좌석 번호 | 이름표 붙은 사물함, 주머니 속 구슬 |

## 정수형 (int, Integer)

**"셀 수 있는 명확한 개수"**

정수는 소수점이 없는 숫자입니다. 일상생활에서 **'사과 1개', '학생 30명', '나이 20살'**처럼 딱딱 끊어지는 수치를 다룰 때 사용합니다.

* **특징:** 아주 큰 숫자도 제한 없이 표현할 수 있습니다.
* **비유:** '알사탕'과 같습니다. 반으로 쪼개지 않은 온전한 알맹이들이죠.

```python
# 정수형 예시
apples = 5          # 양수
zero = 0            # 0
temperature = -10   # 음수

print(type(apples)) # <class 'int'> 출력

```

----------------------------------

## 실수형 (float, Floating-point)

**"끊임없이 이어지는 연속적인 수치"**

실수는 소수점이 포함된 숫자입니다. **'내 키 175.5cm', '시력 1.2', '파이() 3.14'**처럼 정밀한 측정값이 필요할 때 사용합니다.

* **특징:** 소수점 아래 자릿수를 표현하며, 컴퓨터는 이를 '부동 소수점' 방식으로 저장합니다.
* **비유:** '밀가루'나 '물'과 같습니다. 내가 원하는 만큼 아주 미세하게 나누어 측정할 수 있죠.

```python
# 실수형 예시
height = 182.3      # 소수점이 있음
weight = 70.0       # .0이 붙는 순간 정수가 아닌 실수가 됩니다!
pi = 3.141592       # 정밀한 상수

print(type(weight)) # <class 'float'> 출력

```

----------------------------------

## 정수/실수 연산의 주의점

- 정수와 실수가 만나면 어떻게 처리되는지 아는 것이 중요

### 1. 자동 형변환

정수와 실수를 더하면 결과는 항상 **실수(float)**가 됩니다. 더 정밀한 쪽을 선택하는 것이죠.

```python
result = 10 + 5.5   # 10(정수) + 5.5(실수)
print(result)       # 15.5 (결과는 실수)

```
----------------------------------

### 2. 나눗셈의 마법

파이썬에서 나누기(`/`)를 하면, 나누어떨어지더라도 무조건 **실수(float)**를 반환합니다.

```python
print(4 / 2)  # 2가 아니라 2.0이 나옵니다! (실수형)
print(5 / 2)  # 2.5 (실수형)

```

### 3. 몫과 나머지 연산

만약 나눗셈 후 정수 값만 얻고 싶다면 특수 연산자를 사용합니다.

```python
print(5 // 2) # 몫 연산자 (결과: 2) -> 정수형
print(5 % 2)  # 나머지 연산자 (결과: 1) -> 정수형

```

## 진법 표기법

- 숫자 앞에 특정 **'접두사'**를 붙여서 알려준다.

| 진법 | 접두사 | 사용 가능 숫자 | 파이썬 예시 |
| --- | --- | --- | --- |
| **2진법** | `0b` (binary) | 0, 1 | `0b1010` |
| **8진법** | `0o` (octal) | 0 ~ 7 | `0o12` |
| **10진법** | (없음) | 0 ~ 9 | `10` |
| **16진법** | `0x` (hex) | 0 ~ 9, A ~ F | `0xA` |
----------------------------------


- 어떤 진법으로 입력하든 출력할 때는 기본적으로 **10진수**로 계산해서 출력

```python
# 다 같은 숫자 '10'을 표현하고 있습니다.
decimal_num = 10         # 10진수
binary_num = 0b1010      # 2진수 (8 + 2)
octal_num = 0o12         # 8진수 (8*1 + 2)
hex_num = 0xA            # 16진수 (10을 뜻하는 A)

print(decimal_num)  # 10
print(binary_num)   # 10
print(octal_num)    # 10
print(hex_num)      # 10
```

## 불리언(bool)이란?

**"세상을 오직 두 가지로만 나누는 기준"**

- 불리언 자료형은 오직 **`True` (참)**와 **`False` (거짓)** 두 가지 값만 가질 수 있다 
- 논리적인 판단이 필요할 때 사용하며, 제어문(if문, while문)의 핵심 엔진 역할
- '전등 스위치'나 'OX 퀴즈의 정답'과 같다. 중간은 없다!

```python
# 직접 할당
is_active = True
is_finished = False

print(type(is_active))  # <class 'bool'>
```
----------------------------------
## 불리언이 생성되는 비교 연산

- 보통 불리언 값은 직접 입력하기보다, **비교 연산**의 결과로 만들어지는 경우가 많다.

```python
print(10 > 5)   # True (10은 5보다 크다)
print(10 == 5)  # False (10은 5와 같다 - 거짓)
print(10 != 5)  # True (10은 5와 같지 않다 - 참)

# 로그인 체크
user_pw = "1234"
input_pw = "1111"
is_login_success = (user_pw == input_pw)
print(is_login_success)  # False
```
----------------------------------
## 파이썬만의 독특한 개념: "참 같은 값(Truthiness)"

- 파이썬에서는 불리언 자료형이 아니더라도 **"이건 참으로 치자!"** 혹은 **"거짓으로 치자!"**라고 정해놓은 규칙이 있다.

| 데이터 상태 | 불리언 판정 | 예시 |
| --- | --- | --- |
| 빈 상태 | **`False`** | `""` (빈 문자열), `[]` (빈 리스트), `0`, `None` |
| 값이 존재하는 상태 | **`True`** | `"Hello"`, `[1, 2]`, `1`, `-1` |

```python
# 빈 리스트는 False로 간주
basket = []
if not basket:
    print("바구니가 비어 있어요!")

```
